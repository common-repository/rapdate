Rapid checkout after updating or creating pages or posts.

== Description ==

Rapdate is a simple plugin for wordpress sites that need to edit and re-edit a lot of different posts and/or pages. It overcomes the irritating core behavior that binding you to stay inside the post or page after pressing the "update" button.

== Installation ==

Installtion is simple:

1. Upload the directory 'Rapdate' to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. check the checkboxs for Rapid checkout from post and/or pages
